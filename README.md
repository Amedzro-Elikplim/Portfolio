## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Knowledge on basic html and css
- Text Editor (VScode, Atom, Sublime text, etc)

### Setup
- clone repository using `git clone`
- run `npm install`
- run `live server`
